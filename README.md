# nexus-swagger-client
swagger-client for Sonatype Nexus Repository Manager 3

My Nexus OSS version is 3.29.2-02.  

# How to generate swagger-client client and use it
Example you can see in [MinistrBob/MyPythonTools/Nexus](https://github.com/MinistrBob/MyPythonTools/tree/main/Nexus)

Generating a Python client with Swagger.  
```
docker run --rm -v c:\!SAVE\client\:/local swaggerapi/swagger-codegen-cli generate -i http://172.26.12.108:8081/service/rest/swagger.json -l python -o /local/out/python
```
In folder c:\!SAVE\client\out\python\swagger_client\ get a package.  
The package needs to be installed and imported into your project.  
```commandline
-- Create project, create venv and activate it.
cd c:\MyGit\MyPythonTools\Nexus\
py -m venv venv
cd c:\MyGit\MyPythonTools\Nexus\venv\Scripts
activate
c:\MyGit\MyPythonTools\Nexus\venv\Scripts\python.exe -m pip install --upgrade pip
-- Go to client folder
cd c:\!SAVE\client\out\python\
-- installing the generated package in venv
c:\MyGit\MyPythonTools\Nexus\venv\Scripts\python.exe setup.py install
```
